# GetApp

**GetApp** is an Android application for managing your grocery shopping efficiently.

## Features

- **Grocery List Management**: Easily add and organize your shopping items.
- **Smart Formatting**: Automatically formats quantities and units (e.g., "500g", "2x").
- **Dark Mode Support**: Fully optimized for both light and dark themes.

## Download

You can download the latest version of the app from the [Releases](https://github.com/FleximusMaximus/getapp/releases) page.

## Installation

1. Download the `.apk` file from the latest release.
2. Open the file on your Android device.
3. If prompted, allow installation from unknown sources.
